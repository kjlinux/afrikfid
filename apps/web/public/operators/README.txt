Logos des opérateurs de paiement Mobile Money
=============================================

Placez les images dans ce dossier avec les noms exacts suivants :

  orange.png   → Orange Money
  mtn.png      → MTN MoMo
  wave.png     → Wave
  airtel.png   → Airtel Money
  moov.png     → Moov Money / Flooz
  mpesa.png    → M-Pesa

Formats acceptés : PNG, JPG, SVG, WEBP
Taille recommandée : 80×80 px minimum (carré), fond transparent de préférence.

Si une image est absente ou ne charge pas, un logo SVG générique s'affiche automatiquement.
